一、Javascript/css/html

1. What is the difference between null and undefined in JavaScript?

3. What is the difference between CSS Grid and Flexbox?

  • CSS Grid and Flexbox are both layout models in CSS that can be used to create responsive web designs. The main difference between them is that CSS Grid is a two-dimensional layout model, while Flexbox is a one-dimensional layout model.

4. What is React? How does it differ from other JavaScript frameworks?

  • React is a JavaScript library created by Meta (formerly known as Facebook) that helps developers build user interfaces for web pages and native mobile applications. It’s different from other JavaScript frameworks because it uses a **virtual DOM (Document Object Model) **to update the UI efficiently

5. What is the difference between a variable declared with let and a variable declared with var in JavaScript?

  1. let is block-scoped, while var is function-scoped. This means that a variable declared with let is only accessible within the block it was declared in, while a variable declared with var is accessible within the entire function it was declared in.

  2. variables declared with let are not hoisted to the top of their scope like variables declared with var. This means that you can’t use a variable declared with let before it’s been declared

6. What is the difference between the <div> tag and the <span> tag in HTML?

  • The <div> tag is used to create a block-level container that can contain other HTML elements, while the <span> tag is used to create an inline-level container that can contain other HTML elements.

7. What is the box model in CSS?

  1. The box model consists of four parts: content, padding, border, and margin.

  2. There are two types of box models in CSS: the standard box model and the alternative box model.

    • The standard box model is the default one
    • Under the standard box model, the width = the content width
    • Under the alternative box model, the width = the content width + padding + border (but not margin)

8. What is the difference between a block and an inline element in HTML

  1. Block-level elements have their own width and height while inline’s width and height are dependent on their content.
  2. Block-level elements start on a new line and take up the full width available to them by default while inline elements only take up as much width as necessary and do not create a new line.
  3. Block-level elements can include other block or inline elements.

9. What is the difference between HTML and XML?

  • HTML (Hypertext Markup Language) is used to create web pages that can be displayed in a web browser.
  • XML (Extensible Markup Language) is used to store and transport data.

二、React

1. What is the difference between a class component and a functional component in React?

  • Syntax: A class component extends the React.Component class and has a render() method that returns a JSX element, while a functional component is just a plain JavaScript function that returns JSX.
  • State: Class components can have state, which is an object that stores data that can change over time. Functional components can also have state using hooks like useState().
  • Lifecycle methods: Class components have lifecycle methods like componentDidMount() and componentWillUnmount() that are called at different points in the component’s lifecycle. Functional components can use hooks like useEffect() to achieve similar functionality.
  • Performance: Functional components are generally faster than class components because they don’t have the overhead of creating an instance of a class.

2. What is the difference between a controlled component and an uncontrolled component in React?

  1. Controlled components are components that are controlled by React state, while uncontrolled components are components that maintain their own internal state
  2. A controlled component receives its current value and an update callback via props, and the parent component manages the state of the component
  3. An uncontrolled component receives its current value from the DOM

3. What is the difference between state and props in React?

  • state is used to store data that is used within a component. It is mutable and can be changed using the setState method.
  • Props (short for properties) are used to pass data from one component to another. They are immutable and cannot be changed by the component that receives them.
